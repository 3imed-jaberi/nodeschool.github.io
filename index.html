Workshoppers

Workshoppers are the open source lessons themselves. All our self guided (you don't need to attend a workshop to do one) and most work offline.

Get Going —

You’ll need Node.js on your computer to get started with each of these. Then use npm to install each module with the command below it. Once installed, simple type the workshopper’s name to launch.
